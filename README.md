# 🌌 D3xTRverse Flow

### **The ultimate companion for SQL Architects. Visualize. Decode. Dominate.**

![D3xTRverse Flow Hero](./public/hero.png)

> [!IMPORTANT]
> **D3xTRverse Flow** isn't just a visualizer—it's a diagnostic engine. It transforms raw, tangled SQL strings into clear, interactive Directed Acyclic Graphs (DAGs) powered by deterministic AST parsing and high-performance AI insights.

---

## ⚡ The Problem vs. The Solution

**The Problem:** Legacy SQL is a labyrinth. Subqueries nested 5 levels deep, 10-way joins across fragmented schemas, and parallel CTEs make mental modeling a slow, error-prone nightmare. Reading code shouldn't feel like deciphering an ancient scroll.

**The Solution:** Instant, deterministic visualization. **D3xTRverse Flow** parses your SQL using a strict Abstract Syntax Tree (AST), maps out the data flow with mathematical precision, and leverages AI to give you a play-by-play of exactly what's happening at every node.

---

## 🚀 Key Features

*   **🎯 Deterministic Parsing:** 100% accurate node relationships. Handles complex `JOIN` logic, parallel `CTE`s, `UNION`s, and deeply nested subqueries with a strict AST-to-Graph pipeline.
*   **🧠 AI-Powered Insights:** Node-level context generated by **Groq (Llama-3.3-70b-versatile)**. Get instant 1-sentence explanations for every operation without leaving the flow.
*   **👶 "Explain Like I'm 5" (ELI5) Mode:** For those "Wait, what?" moments. Hit the on-demand button to fetch a real-world analogy for complex logic, wrapped in a cinematic terminal loading animation.
*   **📤 Share & Export:** 
    *   **URL State Syncing:** Your entire query encoded via `lz-string`—just copy the URL and share.
    *   **High-Res Canvas Export:** 1-click PNG/SVG export using `html-to-image` for documentation or Slack updates.
*   **🎮 Premium UI/UX:** Built with a dark "Classy Modern Gaming" aesthetic. Features **Monaco/CodeMirror** integration, glassmorphic node styling, and **Framer Motion** for cinematic transitions.
*   **🤡 The "Broke Dev" Easter Egg:** A fake support chat widget that roasts you about Groq API token costs. Try sending a message—if you dare.

## ⚙️ How it Works (Under the Hood)

1.  **Strict Parsing:** Raw SQL is fed into `node-sql-parser` to generate a rigorous AST. 
2.  **Flattening:** A custom utility flattens the AST into a hierarchical `NodeMap`, identifying sources (`FROM`), transformations (`JOIN`, `WHERE`), and outputs (`SELECT`).
3.  **Layout Logic:** The `Dagre` layout engine calculates X/Y coordinates to ensure a collision-free Directed Acyclic Graph (DAG) layout.
4.  **AI Orchestration:** The `NodeMap` is sent to the **Groq API** where Llama-3.3 iterates through each operation to generate human-readable insights.
5.  **Reactive Rendering:** React Flow consumes the layouted nodes and edges, rendering them into a glassmorphic canvas with interactive states.

## 🏁 Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/Saynam221b/dex-floww.git
cd dex-floww
npm install
```

### 2. Environment Setup
Create a `.env.local` file and add your Groq API keys:
```env
# Supports rotation for high availability
GROQ_API_KEY_1=gsk_...
GROQ_API_KEY_2=gsk_...
GROQ_API_KEY_3=gsk_...
```

### 3. Run Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the magic.
